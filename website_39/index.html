AI/DEEP LEARNING WITH CORTANA

Project Overview

Over two million people in the UK are estimated to be currently living with sight loss.

Microsoft Soundscape is an application compatible with iPhone 5s or later. It inform users, especially blind people or people with low vision, about their surroundings using 3D audio cues. Cortana is a virtual assistant that helps you orginize your life, learns over time and interact with plenty of other apps.

Our client is Jarnail Chudge who is innovation architect at Microsoft. He want us to create an application that integrates Soundscape with Cortana. To be more specific, the user will be able to interact with a chatbot so that the user can get suggestions and information about his enviroment. We are also going to introduce Cortana and Maps to our app so that all the appropriate instructions would be available to the user.

Our solution

We decide to use Ionic and Xcode to develop our application. Use Microsoft Bot Framework and Azure Cognitive Services (Speech API, Language understanding API) to create a voice assistant in Soundscape App. Since our client cannot provide us with Soundscape source code or APIs at this point, we can only mock up a Soundscape app, but we will not work much on the functions which the Soundscape already has. Alternatively, we will focus on the new features. So, the “fake Soundscape” will be our platform and we will build an app based on it.

Key Features of the Project

Since our project is to add new features to an existing app called Soundscape, the original features of Soundscape will not be the key features in this project. As an extension of the original app, we add many new features. We build a cloud based chatbot and connect it to our new App. So users can use this voice assistant send command to the app instead of tab on the buttons. The backend will collect users daily behavior and make appropriate recommendations when needed. Users can ask for help when they not sure how and what to do, and the app will provide some useful tips to allow the users learn to use the app.
